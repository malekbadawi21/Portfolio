# Portfolio project

This is my personal portfolio website. Stack: React + Vite, React Router, plain CSS, deployed to Vercel.

Design decisions already made:
- Single font family: Helvetica Neue / Inter
- Color palette: #111111 text, #6b6b6b secondary, #e5e5e5 borders, #ffffff background
- Four columns on the homepage: Roles, Experiences, Projects, Contact
- Each column header is a clickable dropdown — only one open at a time
- Clicking an item loads a detail panel below the table on the same page (no navigation)
- Style: minimalistic, sharp, generous whitespace, uppercase spaced nav labels

Follow the global teaching methodology in this project.
